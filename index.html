Java

Snake Game (class project)

I made my own version of the popular arcade Snake Game using a LinkedList, where each node acted as another connected individual box that made up the snake.


As for the rules, I created methods that would end the game if the snake had run into the wall or itself. Arrow keys were used to move, and if the entire screen filled up with no unwanted collisions the player would recieve an ever so satisfying "You Win!" message.

JavaScript & HTML

Calendar (class project)

This calendar was hosted on a remote EC2 server.
